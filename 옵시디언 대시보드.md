# 📈 섹터 리포트 대시보드

> 이 노트는 [Dataview](https://blacksmithgu.github.io/obsidian-dataview/) 플러그인이 필요합니다.
> 설정 → 커뮤니티 플러그인 → "Dataview" 설치·활성화 후 아래 표가 자동으로 채워집니다.

## 최근 리포트 (30일)

```dataview
TABLE WITHOUT ID
  file.link AS "리포트",
  date AS "날짜",
  sectors AS "섹터"
FROM "reports"
WHERE type = "sector-report"
SORT date DESC
LIMIT 30
```

## 월별 리포트 수

```dataview
TABLE length(rows) AS "리포트 수"
FROM "reports"
WHERE type = "sector-report"
GROUP BY dateformat(date(date), "yyyy-MM") AS "월"
SORT key DESC
```

## 전체 아카이브

```dataview
LIST
FROM "reports"
WHERE type = "sector-report"
SORT date DESC
```

## 활용 팁

- **종목별 히스토리 만들기**: 리포트를 읽다가 관심 종목이 나오면 `[[삼성전자]]`처럼 링크로 바꿔보세요. 그 종목 노트를 열면 언급된 모든 리포트가 백링크로 모입니다.
- **내 메모 덧붙이기**: 리포트 파일에 직접 생각을 추가해도 됩니다. git이 변경 이력을 관리해줍니다.
- **특정 섹터만 보기**: 위 쿼리의 `WHERE` 절에 `AND contains(sectors, "반도체")` 를 추가하면 해당 섹터 리포트만 필터됩니다.
