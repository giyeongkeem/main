🅿️ 어디주차 개인정보 처리방침

최종 수정일: 2026-07-17

수집하는 정보

어디주차는 서버가 없는 앱입니다. 회원가입이 없고, 입력한 주차 위치·사진·메모·GPS 좌표는 전부 사용자의 기기(브라우저 저장소 또는 앱 내부 저장소) 안에만 저장되며 외부로 전송되지 않습니다.

기기 권한 사용 목적

  • 위치(GPS) — 주차한 건물·좌표를 자동 입력하기 위해 사용. 좌표는 기기에만 저장됩니다.
  • 카메라 — 주차장 기둥 표지판 촬영과 층수·구역 자동 인식(기기 내 처리)에 사용됩니다.
  • 동작 센서(기압계) — 지하 몇 층인지 추정하기 위해 사용됩니다.
  • 블루투스 연결 상태(Android) — 차량 블루투스 해제 시 주차 기록을 제안하기 위해 사용됩니다.

제3자 제공

사용자 데이터를 제3자에게 제공하지 않습니다. 웹 버전에서 건물 이름 자동 입력 시 좌표가 OpenStreetMap Nominatim(역지오코딩) 서비스에 조회 목적으로만 전송될 수 있으며, 저장되지 않습니다.

데이터 삭제

앱 안의 "전체 기록 삭제" 또는 앱/브라우저 데이터 삭제로 모든 데이터가 즉시, 완전히 삭제됩니다.
