# 어디주차 🅿️ — 주차 위치 기록 앱

지하 주차장에서 "내 차 몇 층에 세웠더라?"를 해결하는 초경량 웹앱(PWA)입니다.
서버·회원가입 없이 브라우저에만 저장되며, 홈 화면에 추가하면 일반 앱처럼 쓸 수 있습니다.

## 자동 기록 — 무엇이 자동이고 무엇이 아닌가

| 항목 | 자동화 | 방식 |
|---|---|---|
| 건물/장소 이름 | ✅ 자동 | 앱을 열면 GPS를 잡고 역지오코딩(OpenStreetMap Nominatim, 키 불필요)으로 건물 이름을 자동 입력 |
| GPS 좌표 | ✅ 자동 | 기록 화면 진입 시 자동 저장 (권한 허용 시) |
| 층수 / 구역 번호 | ⚠️ 반자동 | 기둥 표지판 사진을 찍으면 OCR(tesseract.js)로 "B3", "C-14" 표기를 자동 인식해 입력 |
| 주차 순간 감지 | ⚠️ 우회 | 웹앱은 블루투스 해제를 직접 감지 못함 → 아래 `?quick` URL로 폰 자동화와 연동 |

**지하 층수의 완전 자동 감지는 웹앱에서는 불가능**합니다. GPS는 지하에서 층을 구분하지 못하고,
네이티브 앱이 층 감지에 쓰는 기압계 센서는 브라우저에 공개 API가 없습니다. 그래서 이 앱은
사진 OCR(온라인 시)로 층수를 자동 인식하고, 오프라인이면 원터치 층수 버튼으로 보완합니다.

### 주차 순간 자동 기록 (`?quick`)

`https://giyeongkeem.github.io/main/?quick` 을 열면 GPS + 건물 이름을 자동으로 채워 **즉시 저장**됩니다.
폰 자동화와 묶으면 차에서 내리는 순간 자동 기록됩니다:

- **iPhone**: 단축어 앱 → 자동화 → "블루투스 — 차량 오디오 연결 해제 시" → "URL 열기: `https://…/?quick`"
- **Android**: Tasker/MacroDroid → 트리거 "블루투스 연결 해제(차량)" → 동작 "URL 열기"

저장 후 앱에서 층수만 확인/수정하면 됩니다(마지막 사용 층수가 기본값).

## 주요 기능

- **지하 층수 기록** — B10 ~ 20F까지 층수 선택. 지하층(B1, B2, …)은 빨간색으로 강조 표시되고, 자주 쓰는 B5~1F는 원터치 버튼 제공. 마지막에 쓴 층수를 기억해 재방문 시 편리합니다.
- **위치 자동 감지** — 앱을 열면 GPS와 건물 이름이 자동으로 채워집니다.
- **사진에서 층수·구역 자동 인식** — 기둥 표지판을 찍으면 OCR로 "B3 · C-14"를 자동 입력(온라인 시. 실패하면 수동 입력으로 자연스럽게 폴백).
- **구역/기둥 번호 + 건물 이름 + 메모** — "C-14 기둥, 엘리베이터 B 근처" 같은 상세 기록.
- **사진 첨부** — 카메라로 기둥 번호를 찍어두면 위치 카드에 함께 표시됩니다(자동 압축 저장).
- **GPS 저장** — 야외/지상 주차 시 위치를 저장하고 카카오맵 링크로 확인.
- **주차 경과시간** — "3시간 24분 경과"를 실시간 표시해 주차 요금 가늠에 도움.
- **공유** — 가족에게 "B3 · C-14" 위치를 메시지로 바로 공유(Web Share API).
- **주차 히스토리** — 종료한 주차 기록을 최근 50건까지 보관.
- **오프라인 동작** — 서비스 워커 캐시로 지하 주차장(통신 불가 환경)에서도 열립니다.
- **다크 모드** — 시스템 설정을 자동 반영.

## 실행 방법

**서비스 주소: https://giyeongkeem.github.io/main/** (푸시하면 자동 재배포)

로컬 실행도 가능합니다:

```bash
cd parking-tracker
python3 -m http.server 8000
# http://localhost:8000 접속
```

배포는 GitHub Pages, Netlify, Vercel 등 정적 호스팅에 폴더째 올리면 끝입니다.
(GPS·카메라·서비스 워커는 HTTPS 환경에서 동작합니다. localhost는 예외적으로 허용.)

### 휴대폰에서 앱처럼 쓰기

1. 배포된 주소를 휴대폰 브라우저로 접속
2. **iOS Safari**: 공유 → "홈 화면에 추가"
3. **Android Chrome**: 메뉴 → "앱 설치" 또는 "홈 화면에 추가"

## 데이터 저장 방식

| 항목 | 저장소 | 비고 |
|---|---|---|
| 현재 주차 위치 | `localStorage` | 사진 포함(JPEG 압축) |
| 지난 기록 | `localStorage` | 최근 50건, 용량 절약을 위해 사진 제외 |
| 마지막 층수 | `localStorage` | 재방문 편의 |

모든 데이터는 기기 브라우저 안에만 저장되며 외부로 전송되지 않습니다.
